Neste aplicativo é possível acessar a lista de produtos cadastrados, adicionar produtos que serão guardados no state e no local storage.
Uma vez criado brinda informações, tais quais, id, nome, marca, e preço. Tendo a opção de eliminar o produto.
A opção editar nao funciona completamente, quando apertar o botão se acessa aos inputs para realizar a edição, porem nao é possível salvar a mesma.
